# 📱 Standard Calculator

A feature-rich, web-based calculator application built with vanilla HTML, CSS, and JavaScript. Perfect for performing basic arithmetic operations and advanced mathematical calculations directly in your browser.

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Basic Operations** | Addition (+), Subtraction (-), Multiplication (*), Division (/) |
| **Advanced Functions** | Factorial (n!), Square (X²), Square Root (²√x), Power of 2 (2^) |
| **Special Operations** | Percentage (%), Reciprocal (1/x) |
| **User Controls** | Clear (Cls), Backspace (×), Decimal point support |
| **Responsive Design** | Works on desktop and tablet devices |
| **Real-time Display** | Instant calculation feedback |

## 🛠️ How to Use Locally

### Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com/saeedahmedbl68-lab/Standard-Calculator.git
   cd Standard-Calculator
   ```

2. Open in your browser:
   - Double-click `index.html`
   - Or right-click → Open with → Browser of choice

### Usage Examples
- **Basic Math**: `5 + 3 =` → Result: `8`
- **Factorial**: `5 n! =` → Result: `120`
- **Square**: `7 X² =` → Result: `49`
- **Square Root**: `16 ²√x =` → Result: `4`
- **Power of 2**: `2^ 3 =` → Result: `8`

### Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Grid layout, flexbox, animations, and gradients
- **Vanilla JavaScript**: No external dependencies

### Key Functions
- `keypress(key)` - Handles numeric input and operators
- `Calculation()` - Evaluates mathematical expressions
- `Clean()` - Clears the display
- `Back()` - Removes the last character
- `Factorial()` - Calculates factorial
- `Xsquare()` - Calculates square of a number
- `SquareRoot()` - Calculates square root
- `OneoverX()` - Calculates reciprocal
- `Exponential()` - Handles power of 2

## 📋 Function Reference

### Basic Operations
- **+** → Addition
- **-** → Subtraction
- ***** → Multiplication
- **/** → Division

### Advanced Functions
- **n!** → Factorial (e.g., 5! = 120)
- **X²** → Square (e.g., 5² = 25)
- **²√x** → Square Root (e.g., √25 = 5)
- **2^** → Power of 2 (e.g., 2^3 = 8)

### Special Operations
- **%** → Percentage/Modulo
- **1/x** → Reciprocal (e.g., 1/5 = 0.2)

### Controls
- **Cls** → Clear display
- **×** → Backspace (delete last digit)
- **.** → Decimal point
- **=** → Calculate result
